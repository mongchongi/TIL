# 비교 연산자

```javascript
// > : 좌항이 우항보다 크면 true, 작으면 false
console.log(2 > 1); // true

// < : 좌항이 우항보다 작으면 true, 크면 false
console.log(2 < 1); // false

// >= : 좌항이 우항보다 크거나 같으면 true, 작으면 false
console.log(3 >= 3); // true

// <= : 좌항이 우항보다 작거나 같으면 true, 크면 false
console.log(3 <= 3); // true

// == : 좌항과 우항이 같으면 true, 다르면 false (동등)
console.log(1 == 1); // true
console.log(1 == '1'); // true

// != : 좌항과 우항이 다르면 true, 같으면 false (부등)
console.log(1 != 1); // false
console.log(1 != '1'); // false

// === : 좌항과 우항이 값과 타입까지 같으면 true, 다르면 false (일치)
console.log(1 === 1); // true
console.log(1 === '1'); // false

// !== : 좌항과 우항이 값과 타입까지 다르면 true, 같으면 false (불일치)
console.log(1 !== 1); // false
console.log(1 !== '1'); // true
```
